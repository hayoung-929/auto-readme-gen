## 시작하기


### 설치 방법

**1. 저장소 복제**
```bash

git clone https://github.com/hayoung-929/auto-readme-gen.git

cd auto-readme-gen
```

**2. 가상환경 설정**
```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
```

**3. 패키지 설치**
```bash
pip install -r requirements.txt
```
